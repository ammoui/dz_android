## Тема

Приложение для ведения личной книги рецептов с управлением статусом: хочу приготовить -> готовлю -> готово.

## Что умеет

1. Поиск рецептов по названию или категории.
2. Фильтрация по статусам (все / хочу приготовить / готовлю / готово).
3. Изменение статуса рецепта на экране деталей.
4. Подсчёт статистики по текущей выборке (всего, хочу, готовлю, готово).
5. Полностью офлайн: данные лежат в коде, без сети и БД.
