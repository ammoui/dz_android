## Выбранный API

- Jikan API v4: https://api.jikan.moe/v4/

## Что дает API

- Данные об аниме и манге без ключа и без регистрации.
- Поиск по названию, список популярных тайтлов и экран деталей по id.

## Как запустить проект

1. Открыть проект в Android Studio.
2. Дождаться синхронизации Gradle.
3. Запустить модуль app на эмуляторе или устройстве с интернетом.
